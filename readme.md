# Goddis
A Partial clone of redis in GO.

##### Implemented commands
- PING
- ECHO
- EXISTS
- EXPIRE
- DEL
- SET (EX,PX,NX,XX)
- GET
- INCR
- INCRBY
- MGET
- HSET
- HGET
- HMGET
- HINCRBY

Doesn't respond to INFO
